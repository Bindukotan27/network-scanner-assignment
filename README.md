# Network Scanner Assignment

## Overview
This project implements network scanning tools using Python. It includes Ping scanning, ARP table retrieval, and Nmap-based scanning.

## Files

### ping_scanner.py
Checks whether a host is reachable and measures response time.

### arp_scanner.py
Displays ARP table showing IP and MAC address mappings.

### nmap_scanner.py
Performs host discovery, port scanning, service detection, and OS detection.

## How to Run

Open terminal in the project folder and run:

python ping_scanner.py  
python arp_scanner.py  
python nmap_scanner.py
